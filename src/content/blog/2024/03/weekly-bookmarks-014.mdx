A bit later than usual, but I did finally read all the articles and posts about CSS that I bookmarked in the last week.

- [“Modern CSS Tooltips And Speech Bubbles (Part 1)”](https://www.smashingmagazine.com/2024/03/modern-css-tooltips-speech-bubbles-part1/) by [Temani Afif](https://css-articles.com/) — an article about a technique to implement tooltips with `border-image` and `clip-path` (and a bunch of math). I wish there was some native way to do some of this (maybe I should come up with a proposal for this or something).

- [“Going beyond pixels and (r)ems in CSS - Container query length units”](https://techhub.iodigital.com/articles/going-beyond-pixels-and-rems-in-css/container-query-length-units) by [Brecht De Ruyte](https://utilitybend.com/) — a practical article about using container query length units.

- [“The New CSS Math: pi and other constants”](https://danielcwilson.com/posts/mathematicss-constants/) by [Daniel C. Wilson](https://danielcwilson.com/) — an article about the new math constants: `pi`, `e`, `infinity`, and even `NaN`.

- [“CSS Foundations: What is IACVT?”](https://www.bram.us/2024/02/26/css-what-is-iacvt/) by [Bramus](https://www.bram.us/) — a post explaining what “Invalid At Computed Value Time” is in CSS. A good read in case you want to understand how things work on a more foundational level.

- [“Case Study: lynnandtonic.com 2023 refresh”](https://lynnandtonic.com/thoughts/entries/case-study-2023-refresh/) by [Lynn Fisher](https://lynnandtonic.com/) — a detailed case study about another annual Lynn’s redesign of her website. A lot of [ResizeObserver](https://developer.mozilla.org/en-US/docs/Web/API/ResizeObserver) usage, CSS variables, and nice tiny details.

- [“Retrofitting fluid typography”](https://clagnut.com/blog/2428/) by [Richard Rutter](https://clagnut.com/) — a case study of using [utopia.fyi](https://utopia.fyi/) to implement fluid typography with custom CSS properties.

- [“Menus, toasts and more with the Popover API, the dialog element, invokers, anchor positioning and `@starting-style`”](https://frontendmasters.com/blog/menus-toasts-and-more/) by [Ollie Williams](https://fullystacked.net/) — an article about all the new features we will have in the future to work with popovers and dialogs which outlines many aspects we need to consider when implementing them.

- [“The value of live web design”](https://chenhuijing.com/blog/the-value-of-live-web-design/) by [Chen Hui Jing](https://chenhuijing.com/) — a post about designing live in the browser, and a list of CSS properties that might be helpful to learn and apply when doing it.

- [“CSS-only bottom-anchored scrolling area”](https://kittygiraudel.com/2024/02/26/css-only-bottom-anchored-scrolling-area/) by [Kitty Giraudel](https://kittygiraudel.com/) — a short post about two ways of making scrolling containers start from bottom. One of them was new to me: I did not know that using `flex-direction: column-reverse` did work like that!

- [“An HTML Switch Control”](https://webkit.org/blog/15054/an-html-switch-control/) by [Anne van Kesteren](https://annevankesteren.nl/) and [Jon Davis](https://github.com/jdatapple) — a post about the new switch control introduced in Safari 17.4.
